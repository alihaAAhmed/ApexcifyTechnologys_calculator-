# Calculator

A simple, fully functional calculator built with HTML, CSS, and JavaScript.

## Live Demo

Visit Website: https://alihaaahmed.github.io/ApexcifyTechnologys_calculator-/

## Features

- All arithmetic operations (+, -, ×, ÷)
- Percentage calculation (%)
- Clear screen (AC)
- Delete last character (DEL)
- Decimal point support
- Real-time result display
- Responsive design for all devices
- Keyboard support (Number keys, Enter, Backspace, Escape)
- Hover effects and smooth transitions
- Zero division error handling

## Keyboard Shortcuts

| Key | Function |
|-----|----------|
| 0-9 | Enter numbers |
| . | Decimal point |
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| % | Percentage |
| Enter | Calculate result |
| Backspace | Delete last character |
| Escape | Clear all |

## Tech Stack

- HTML5 - Structure
- CSS3 - Styling and animations
- JavaScript (ES6) - Logic and keyboard events
